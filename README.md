# PDF-based-question-answering
Offline AI system for asking questions from PDFs using OCR, FAISS and HuggingFace models.
Requirements:
- Python 3.10+
- For scanned PDFs on Windows: Install Poppler
  https://github.com/oschwartz10612/poppler-windows/releases/

Install:
python -m pip install -r requirements.txt
python main.py
